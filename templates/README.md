# Templates

This directory contains templates for consistent documentation across the project.

## Available Templates

### Engineering Documentation
- **`calculation-notebook-template.ipynb`** - Jupyter notebook for engineering calculations
- **`research-notes-template.md`** - Documenting research and literature review
- **`verification-report-template.md`** - Verifying calculations and designs

### Design Documentation
- **`design-iteration-template.md`** - Documenting design iterations and changes

### Testing Documentation
- **`test-results-template.md`** - Documenting test results and analysis

### Manufacturing Documentation
- **`manufacturing-process-template.md`** - Step-by-step manufacturing procedures

## Usage

1. Copy the appropriate template to your working directory
2. Rename with descriptive name
3. Fill in all sections
4. Remove placeholder text
5. Keep "TBD" for sections you'll complete later

## Template Locations

Templates are organized by purpose:
- Calculations → `calculation-notebook-template.ipynb`
- Testing → `test-results-template.md`
- Design → `design-iteration-template.md`
- Manufacturing → `manufacturing-process-template.md`
- Research → `research-notes-template.md`
- Verification → `verification-report-template.md`

See `docs/decisions/` for the decision record template (DEC-NNN format).
