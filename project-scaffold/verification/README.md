# Verification Directory

Engineering verification and validation.

## Structure

```
verification/
├── hand-calcs/        # Manual calculation checks
├── cross-checks/      # Alternative method verification
└── validation-tests/  # Test-based validation
```

## Verification Methods

1. **Hand Calculations**: Independent manual check
2. **Cross-checks**: Different tool/method same answer
3. **Validation Tests**: Physical testing confirms analysis

## Workflow

1. Identify critical calculations
2. Choose verification method
3. Document in appropriate folder
4. Link back to original calculation

Use `templates/verification-report-template.md`
