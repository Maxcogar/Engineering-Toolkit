# Documentation Directory

Project documentation organized by type.

## Structure

- `decisions/` - Engineering decision records (DEC-NNN format)
- `reference/` - PDFs, datasheets, external documentation
- `research/` - Research notes and findings
- `systems/` - System-level documentation

## Adding Documentation

1. **Decisions**: Use `/decision` command or copy `decisions/000-template.md`
2. **Research**: Use `/research` command or copy template from `templates/`
3. **Systems**: Use `/understand` command to create system documentation

## Naming Conventions

- Decision records: `DEC-001-topic-name.md`
- Research docs: `topic-research.md`
- System docs: `system-name.md`
