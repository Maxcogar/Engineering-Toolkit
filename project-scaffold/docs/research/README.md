# Research Documentation

Research findings, literature reviews, and investigation notes.

## Creating Research Documents

1. Use `/research [topic]` command in Claude Code
2. Or copy template from `templates/research-document-template.md`

## Structure

Research documents should include:
- Objective and scope
- Sources consulted
- Key findings
- Implications for the project

## Files

- `_sources.md` - Track research sources and URLs
- `topic-research.md` - Individual research documents
