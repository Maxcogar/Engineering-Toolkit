# Manufacturing Directory

Manufacturing documentation, processes, and BOM.

## Structure

```
manufacturing/
├── processes/    # Process documentation
├── bom.md       # Bill of materials
└── toolpaths/   # CNC, 3D print files
```

## Bill of Materials

Track all parts in `bom.md` using the template.

## Process Documentation

Document each manufacturing process:
- Setup requirements
- Step-by-step procedure
- Quality checks
- Lessons learned
